USB 3.0 和 USB 2.0 都是通用串行总线（Universal Serial Bus）的标准版本，它们在多个方面存在差异。 一、外观区别 颜色标识： USB 2.0 接口通常为黑色或白色。 USB 3.0 接口通常为蓝色，一些厂商也可能采用其他颜色进行区分，但蓝色较为常见。 接口形状： 从外观上看，USB 3.0 和 USB 2.0 的接口形状相似，但 USB 3.0 接口通常会多一些额外的触点。 二、性能差异 数据传输速度： USB 2.0 的理论最高传输速度为 480Mbps（兆比特每秒）。 USB 3.0 的理论最高传输速度可达 5Gbps（吉比特每秒），相比 USB 2.0 有了极大的提升。这使得在传输大文件，如高清视频、大型游戏等时，USB 3.0 能够显著缩短传输时间。 供电能力： USB 2.0 提供的最大电流一般为 500mA。 USB 3.0 能够提供更大的电流，通常为 900mA，这意味着它可以为连接的设备提供更强大的供电能力，例如可以更快地为移动设备充电，或者支持一些需要更高功率的外部设备。 三、兼容性 向下兼容： USB 3. ...

新增摄像头
发表于2024-08-22
如何新增摄像头（安装好之后的操作）①首先使用设备搜索软件进行局域网范围搜索，可以看到有未激活的摄像头，一般ip为192.168.1.64，激活设备，然后在此软件内修改设备ip，修改大段ip，例如：192.168.99.232，修改好之后就在IVMS软件中进行测试，在维护与管理中添加设备ip，可以看到设备的状态是在线的说明添加成功，然后在主预览中可以看到画面，看是否添加的是同一个摄像头，然后修改ip，加入硬盘录像机，看硬盘录像机是什么段的ip就修改什么段ip。 例如上面这个摄像头的ip是192.168.102.13，所以修改设备ip为192.168.13.77，此硬盘录像机通道数为64，也就是说只能存放64个摄像头的录像，修改好设备ip之后就进入硬盘录像机页面的通道管理，把设备ip加进去。加进去之后要做平台做同步才能加入对应的区域。 然后到平台把摄像头加入到对应的区域即可。

监控ip修改
发表于2024-08-19
新增监控ip一般为192.168.1.64需要电脑和交换机直连或者工程宝和设备所连交换机直连，然后再网页输入192.168.1.64可以进入新安装的摄像头页面，点击配置，点击网络，点击ip，修改ip，同时修改网关，最后点击确定，重启摄像头即可。
